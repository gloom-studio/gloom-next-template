# create-gloom

CLI for scaffolding Gloom templates.

## Usage

```bash
pnpm create gloom
pnpm create gloom next
pnpm create gloom next my-app
```

Current templates:

- `next`

The CLI currently pulls from `gloom-studio/nextjs-starter-gloom`. As more templates are added, extend `TEMPLATE_SOURCES` in `src/index.mjs`.
